💡 Acerca de May

May es un asistente inteligente diseñado para ayudarte a validar, corregir y entender tu código Java en tiempo real usando inteligencia artificial.

🚀 Tecnologías utilizadas:

  • Angular + TailwindCSS
  • Highlight.js para resaltado de sintaxis
  • OpenAI API para evaluación de código
  • TypeScript y estructura modular standalone
